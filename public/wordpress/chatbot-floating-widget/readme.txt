== Description ==

Chatbot Floating Widget is a powerful WordPress plugin that adds an intelligent floating chat widget to your website. Connect with your existing chatbot service to provide real-time customer support, lead generation, and user engagement.

### Key Features

* **Floating Chat Widget** - Beautiful, customizable chat widget that floats on your site
* **AI-Powered** - Connect with advanced AI chatbot services for intelligent conversations
* **Mobile Responsive** - Works perfectly on all devices and screen sizes
* **Customizable Themes** - Match the widget to your brand colors and style
* **Multiple Positions** - Place the widget in any corner of your site
* **Advanced Settings** - Control where and when the widget appears
* **Analytics Ready** - Built-in support for conversation analytics
* **Easy Setup** - Simple configuration with live preview
* **WordPress Integration** - Seamlessly integrates with WordPress admin panel

== Installation ==

1. Download the plugin files from the WordPress repository or from the author's website
2. Upload the entire `chatbot-floating-widget` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' screen in WordPress admin
4. Configure the widget by going to **Settings â†’ Chatbot Widget**
5. Enter your chatbot API URL and API Key
6. Customize the appearance and behavior to match your needs
7. Save your settings and the widget will appear on your website

== Configuration ==

### Basic Setup

1. **API URL**: Enter your chatbot service endpoint URL
2. **API Key**: Provide your authentication API key
3. **Widget Position**: Choose where the widget appears (bottom-right, bottom-left, top-right, top-left)
4. **Enable Widget**: Turn the widget on or off across your site

### Widget Settings

* **Widget Title**: Customize the header title of the chat widget
* **Welcome Message**: Set the initial greeting message
* **Input Placeholder**: Customize the text input placeholder
* **Show on Mobile**: Enable/disable widget on mobile devices
* **Auto Open**: Automatically open the widget when page loads

### Theme Customization

* **Primary Color**: Main color for headers and user messages
* **Button Color**: Color of the floating chat button
* **Text Color**: Text color for chat headers and user messages

### Advanced Options

* **Exclude Pages/Posts**: Hide widget on specific content
* **Load Delay**: Delay widget loading for performance optimization
* **Debug Mode**: Enable console logging for troubleshooting

== Frequently Asked Questions ==

= How do I get an API Key? =

You need to sign up for a chatbot service provider to get your API URL and API Key. Check the plugin documentation for recommended providers.

= Can I customize the widget appearance? =

Yes! The plugin includes extensive customization options for colors, position, messages, and behavior. You can also add custom CSS for advanced styling.

= Does this work with any theme? =

Yes, the plugin is designed to work with any WordPress theme. It uses modern CSS and JavaScript that are compatible with all major themes.

= Is the widget mobile-friendly? =

Absolutely! The widget is fully responsive and adapts to all screen sizes. You can also choose to hide it on mobile devices if needed.

= Can I track chat conversations? =

The plugin includes built-in support for analytics. You can integrate with Google Analytics or other tracking services to monitor chat performance.

= Does this slow down my website? =

No, the plugin is optimized for performance with lazy loading options and efficient code. You can also set a load delay to further optimize page load times.

== Developers ==

**Hooks and Filters:**
The plugin provides several hooks for developers:

* `chatbot_widget_config` - Filter widget configuration
* `chatbot_widget_before_render` - Action before widget renders
* `chatbot_widget_after_render` - Action after widget renders
* `chatbot_widget_message_sent` - Action when message is sent

**Template Override:**
You can customize the widget appearance by overriding templates in your theme.
